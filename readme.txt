A lightweight SEO plugin that automatically adds meta description tags and JSON-LD structured data to your WordPress posts.

== Description ==

This plugin adds basic SEO enhancements to your WordPress website:
- Automatically generates a meta description tag based on the post excerpt.
- Adds JSON-LD schema (BlogPosting) to improve search engine indexing.
- Lightweight and fast. No settings required. Just install and activate.

Ideal for minimal blogs, SEO testing, and educational purposes.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/wp-simple-seo-tools`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Thatâ€™s it! No configuration needed.

== Frequently Asked Questions ==

= Can I customize the meta description? =
Yes, just write a custom excerpt in your post editor.

= Does it support pages? =
Currently, it only adds schema and meta for single posts.
